# 05-1. 함수의 기본 형태

> 함수 호출 === 함수를 사용하는 것
> 매개 변수 : 함수를 호출 할 때 괄호 내부에 넣는 여러 가지 자료
> 리턴값 : 함수를 호출해서 최종적으로 나오는 결과

## 익명 함수

함수란? 코드의 집합을 나타내는 자료형

```
function () { }
```

함수의 좋은 점?

- 반복되는 코드의 경우, 한 번의 정의로 반복적으로 사용 가능 => 반복 작업을 피할 수 있음
- 긴 프로그램을 기능별로 여러 함수로 나눠 작성하면 모듈화 가능 => 코드의 가독성이 좋아짐
- 기능별(함수별)로 수정 가능 => 유지 보수 편리
